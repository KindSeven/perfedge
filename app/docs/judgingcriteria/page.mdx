# 性能评判标准

通篇来讲，我们说性能优化，那么至少在优化之前，我们需要知道一些性能评判标准。这一点的意义在于，我们有了性能评判标准，才能知道我们需不需要优化、才能知道我们优化了没有、优化到什么程度、以及如何进一步优化。

## 本章节内容

- **分析工具** ： 目前市面上有很多现代化的分析工具，可以帮助我们分析性能，比如 Chrome DevTools、Lighthouse、WebPageTest 等。这些工具可以帮助我们找到性能瓶颈，并给出优化建议，我们讲着重介绍这些工具的使用，理解一些常见的性能指标。
- **直观感受** ： 前端技术最终呈现是 UI 界面。因此，除了分析工具之外我们还可以通过直观的感受来判断性能的好坏，比如页面加载速度、页面响应速度、页面流畅度等。这些直观的感受可以帮助我们快速判断性能的好坏，并指导我们进行优化。
